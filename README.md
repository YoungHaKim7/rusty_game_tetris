# rusty_game_tetris

```
cargo run
```

# key

||key|
|-|-|
|rotate |arrow up ( ↑ )|
|left or right | ( ← )  or ( → ) |
|Drop a block quickly | space ( ␣ ) or arrow down (↓) |
|-|-|
|exit game| ESC |
